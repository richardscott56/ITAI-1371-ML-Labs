# ITAI-1371-ML-Labs
Overview

This repository contains laboratory exercises for ITAI-1371: Introduction to Machine Learning.
The labs are primarily completed using Google Colab and Python, with GitHub used for version control and assignment submission.

Purpose

The purpose of this repository is to:

Store and organize lab assignments completed in Google Colab
Practice Python programming in a cloud-based environment
Learn how GitHub integrates with Jupyter notebooks and Colab
Maintain a permanent record of lab work for the course

Google Colab Usage

Most labs in this repository are developed using Google Colab, which allows Python notebooks to be run without local software installation.

Repository Structure

Labs will be organized by lab number or topic.
Each lab folder may contain:

Jupyter notebooks (.ipynb)
Associated files or datasets
Notes related to the lab
